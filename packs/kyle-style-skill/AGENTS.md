# Project Agent Instructions

## Writing style priority

When the task involves rewriting, polishing, formalizing, simplifying, or generating technical/academic/business writing, prefer the `kyle-style-rewriter` skill.

Trigger phrases include:

- 用我的语言习惯表达
- 让我们润色一下
- 说人话
- 去掉 AI 味
- 改成正式文档风格
- rewrite in my style
- make this less AI-generated

## Default mode

Use `strict` mode by default for:

- papers
- proposals
- course materials
- patent drafts
- technical reports
- customer-facing formal documents

Use `normal` mode for general polishing.
Use `light` mode only when the user explicitly asks for minor edits.

## Output quality requirements

The output should be:

- structured
- concise
- logically connected
- restrained in tone
- free from unnecessary rhetoric

Avoid:

- marketing language
- slogan-like endings
- excessive metaphors
- unsupported claims
- overlong sentences
